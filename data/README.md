# Data

This project expects the HMEQ home-equity loan dataset to be placed at:

```text
data/hmeq.csv
```

The raw CSV is intentionally excluded from version control by `.gitignore` until the dataset's redistribution terms are confirmed. If you obtained the dataset from a course platform or another restricted source, do not upload it publicly.

The HMEQ dataset is widely used as a public credit-risk sample dataset, often associated with SAS/Enterprise Miner examples. If you decide to include the data in the repository, cite the original source clearly in the README and confirm that redistribution is allowed.
